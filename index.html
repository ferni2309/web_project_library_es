Los preferidos de nuestros lectores

Que nuestros lectores y lectoras aman

  • El programador pragmático: De la codificación al éxito de Andrew Hunt y David Thomas.

    Este libro es una guía practica para mejorar tus habilidades como programador. Se enfoca en la aplicación de principios y técnicas para escribir código de alta calidad, trabajar de manera efectiva en equipo y adaptarse a los cambios en el entorno de desarrollo.

    David Thomas, Andrew Hunt.

  • Desarrollo de software ágil con Scrum.

    Este libro, escrito por Ken Schwaber, es una guía práctica para aplicar la metodología Scrum en el desarrollo de software ágil. Se enfoca en la aplicación de principios y técnicas para trabajar de manera efectiva en equipo y entregar software de alta calidad de manera rápida y flexible.

    Ken Schwaber y Mike Beedle.

  • Introducción a la Programación con Pyton.

    Es un libro, escrito por Eric Matthes que se enfoca en eseñar a los lectores a programar con Python de manera práctica y accesible. El libro es ideal para principiantes que buscan aprender a programar desde cero o para aquellos que desean aprender Python como segundo leguaje de programación.

    Eric Matthes

  • Inteligencia Artificial: Un enfoque moderno.

    Este libro, escrito por Stuart Russell y Peter Norving, es una guía completa sobre inteligencia artificial. Se enfoca en la aplicación de principios y técnicas para desarrollar sistemas inteligentes que puedan percibir, razonar y actuar en entonrnos complejos.

    Stuart Russell, Peter Norving.
